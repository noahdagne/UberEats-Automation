### Code Workflow

* `first.new_phone(self)`

  * Generates a non-VOIP number to verify codes in the future

  * Handles 2FA confirmation

  * Enter received SMS code to console

* `first.sign_up(self)`

  * Generates random name to be used & inputs into corresponded boxes
